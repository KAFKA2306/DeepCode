# ユーティリティ (Utils)

このディレクトリは、プロジェクト全体で使用される汎用的なヘルパー関数やクラスを提供します。

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `cli_interface.py` | CLI用の共通インターフェース処理（`cli/`内のものとは別に、共通ライブラリとして機能）。 |
| `cross_platform_file_handler.py` | OS間の差異（パス区切り文字など）を吸収するファイル操作ハンドラー。 |
| `dialogue_logger.py` | エージェントとユーザー、またはエージェント間の対話ログを記録・保存するロガークラス。 |
| `file_processor.py` | テキストファイルの読み書き、エンコーディング処理、正規化などを行うユーティリティ。 |
| `llm_utils.py` | LLM（Large Language Model）呼び出しのラッパー。APIキー管理、モデル選択、リトライロジックなどを抽象化。 |
| `simple_llm_logger.py` | LLMへの入出力を簡易的に記録するためのロガー。デバッグ用。 |

## 役割

具体的なビジネスロジックを持たず、複数のコンポーネントから参照される共通機能を提供します。コードの重複を防ぎ、保守性を高めるために利用します。
