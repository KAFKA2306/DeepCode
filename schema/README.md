# スキーマ (Schema)

このディレクトリは、DeepCodeで使用される設定ファイルやデータ構造の検証用スキーマを格納します。

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `mcp-agent.config.schema.json` | `mcp_agent.config.yaml` ファイルの構造を定義するJSONスキーマです。設定ファイルのバリデーションに使用されます。 |

## 役割

プロジェクト全体の設定の一貫性と正確性を保証するために使用されます。ユーザーが誤った設定値を入力した場合に、このスキーマに基づいて早期にエラーを検出します。

## 使用方法

このスキーマは、VS Codeなどのエディタで `mcp_agent.config.yaml` を編集する際の補完やバリデーションにも利用可能です。設定ファイルの先頭で `$schema` プロパティとして指定することで有効になります。
