# CLI (コマンドラインインターフェース)

このディレクトリは、DeepCodeのコマンドラインインターフェース（CLI）の実装を含みます。
CLIは、ターミナル環境でDeepCodeの機能 ("DeepCode: Open Agentic Coding") を利用するための主要なエントリーポイントです。

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `main_cli.py` | **エントリーポイント**。CLIアプリケーションの起動スクリプトです。引数の解析とメインループの開始を担当します。 |
| `cli_app.py` | CLIアプリケーションのコアロジック。ユーザー入力の処理、エージェントワークフローの呼び出し、状態管理を行います。 |
| `cli_interface.py` | ユーザーインターフェース（UI）の描画と対話を担当します。Richライブラリを使用して、ターミナル上で視覚的な出力を提供します。 |
| `cli_launcher.py` | 環境設定や初期化処理を含む、CLIのランチャースクリプトです。 |
| `workflows/` | CLI固有のワークフロー定義が含まれるサブディレクトリです。 |

## 使用方法

### 起動

プロジェクトのルートディレクトリから以下のコマンドでCLIを起動します：

```bash
# UVを使用する場合（推奨）
uv run python cli/main_cli.py

# 通常のPythonの場合
python cli/main_cli.py
```

### 主な機能

- **インタラクティブモード**: ユーザーとの対話形式で、研究論文の解析からコード生成までをガイドします。
- **論文解析**: 指定された論文（要約、URL、ファイルパス）を読み込み、構造化された情報を抽出します。
- **コード生成**: 解析結果に基づき、実装計画の策定、コードの生成、テストの実行を行います。
- **進捗表示**: 各ステップの実行状況をリアルタイムで視覚的に表示します。

## 開発者向け情報

新しいCLIコマンドや機能を追加する場合は、主に以下のファイルを編集します：

1.  **コマンドの追加**: `cli_app.py` にハンドラーを追加します。
2.  **表示の変更**: `cli_interface.py` のレンダリングロジックを修正します。
3.  **引数の変更**: `main_cli.py` の `argparse` 定義を更新します。
