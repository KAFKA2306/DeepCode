# 設定 (Config)

このディレクトリは、DeepCodeのエージェントシステムで使用されるツール定義と設定関連のロジックを含みます。
（注: メインの設定ファイル `mcp_agent.config.yaml` はプロジェクトルートに配置されています）

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `mcp_tool_definitions.py` | MCP (Model Context Protocol) サーバーで使用可能なツールの定義を含みます。使用する各ツール（検索、ファイル操作など）のスキーマとメタデータを管理します。 |
| `mcp_tool_definitions_index.py` | コードインデックス作成や参照に関連する特定のツール定義を管理します。 |

## 役割

このディレクトリのモジュールは、エージェントが外部環境と対話するための「手足」となるツールのインターフェースを定義します。エージェントはこれらの定義を参照して、どのような操作が可能か（例：Web検索、ファイル読み書き）を理解します。

## 設定の変更

新しいツールを追加する場合：

1.  `mcp_tool_definitions.py` に新しいツールのクラスまたは関数定義を追加します。
2.  適切なスキーマ（引数の型、説明）を定義し、エージェントが正しくツールを呼び出せるようにします。
