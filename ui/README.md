# ユーザーインターフェース (UI)

このディレクトリは、Streamlitを使用したDeepCodeのWebアプリケーションのソースコードを含みます。

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `streamlit_app.py` | **エントリーポイント**。Webアプリケーションのメインスクリプトです。画面レイアウトの構築とメインループを開始します。 |
| `components.py` | UIコンポーネント（チャットボット表示、ファイルアップローダー、ボタンなど）の定義。再利用可能なウィジェット。 |
| `handlers.py` | ユーザーのアクション（ボタンクリック、フォーム送信など）に対するイベントハンドラー。バックエンドロジックとの橋渡しを行います。 |
| `layout.py` | アプリケーション全体のページレイアウト定義。 |
| `sidebar_feed.py` | サイドバーのコンテンツ（設定、履歴など）の描画ロジック。 |
| `styles.py` | カスタムCSS定義。アプリケーションの見た目やテーマを管理します。 |

## 役割

ユーザーに視覚的で直感的な操作環境を提供します。複雑なエージェントの動作状況や生成されたコード、実行結果を分かりやすく表示する役割を担います。

## 起動方法

ルートディレクトリから以下のコマンドを実行します：

```bash
uv run streamlit run ui/streamlit_app.py
```

ブラウザが自動的に開き、`http://localhost:8501` にアクセスします。
