# プロンプト (Prompts)

このディレクトリは、DeepCodeのAIエージェントで使用されるシステムプロンプトとテンプレートを管理します。

## ディレクトリ構造

| ファイル名 | 説明 |
|:---|:---|
| `code_prompts.py` | 各エージェント（オーケストレーター、コーダー、レビュアーなど）に与える役割定義、指示、制約条件を含むプロンプト文字列を定義しています。 |

## 役割

プロンプトエンジニアリングの中核となるディレクトリです。エージェントの振る舞い、出力の形式、思考プロセス（Chain-of-Thought）はこのディレクトリ内の定義によって制御されます。

## 開発者向け情報

エージェントの挙動を調整・改善する場合は、`code_prompts.py` 内の該当するプロンプト定数を編集します。

- **役割定義**: エージェントの専門性（例：「あなたはシニアソフトウェアエンジニアです」）を定義。
- **タスク指示**: ツールの使用方法、コーディング規約、エラー処理の方針などを指示。
- **出力形式**: JSONやMarkdownなど、期待する出力構造を指定。
